Maven is a build automation and project management tool primarily used in Java projects.
It helps developers manage project builds, dependencies, documentation, and deployment in a consistent and automated way.

Dependency Management

Maven automatically downloads required JAR files and plugins from a central repository.

You just declare them in a file (called pom.xml).

Example:

<dependencies>
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-web</artifactId>
        <version>3.2.0</version>
    </dependency>
</dependencies>


Build Lifecycle
Maven provides predefined phases like:

compile ‚Üí compiles source code

test ‚Üí runs unit tests

package ‚Üí creates a JAR/WAR file

install ‚Üí installs to local repository

deploy ‚Üí deploys to remote repository

Example:

mvn clean install
